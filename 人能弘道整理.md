## ## 读书《人能弘道：傅佩荣谈论语》

孔子的道就是[仁](#T4)（人生的过程与目的），通过行仁的过程，最后达成天命。孔子每个十年有一种感悟，其上领于天命（被动），下达于使命（主动），“七十从心所欲不逾矩”，表明孔子不太认可朱熹、王阳明的心即良知。读罢傅佩荣老师的解读，可以看出孔子的终极信仰来自于天，“获罪于天，无所祷也”，而在实际中则体现了孔子“内心真诚”“人性向善”“择善固执”“承礼启仁”“人文化成”的理想。

人生在世的行为判断，分三个阶段，六个层次（**孔子的价值观**）：

- 第一阶段是自我中心，其价值分为生存（小人怀土）和发展（小人怀惠）【*放到国与国之争就是丛林法则之争？*】
- 第二阶段是人我互动，其价值分为礼法（君子怀刑）和情义（君子怀德）
- 第三阶段是超越自我，其价值分为无私和止于至善。

个人修身其基本原则是心怀[真诚](#T17)，时刻自省，成为有[德](#T11)之[士](#T10)；然后结交[朋友](#T6)，与人交往讲忠信；最后用礼乐来实现一组人的和谐，扩展到[从政](#T8)要遵循的基本原则。先自我要求，敬事立信；再兼顾节用与爱人，[知人任事](#T5)，最后要多为老百姓着想。礼和乐都是求仁的手段，[礼](#T16)是用于区分的规定，其目的是层次；重视追求心灵的沟通，其目的是和谐。个人修身表现在各个方面，参见[君子与小人](#T1)一节。

[**善**](#T9)是人与人之间适当关系的实现，包括内心感受，人与人对彼此的要求，人与人之间的规范（也就是礼，礼是为了使原有的美质展现，如易经中“白贲无咎”）。孔子评价事物的层级由高到低分别是“**善（人性向善）、美（指善的效果）、言（值得称赞的德行）、蔽（遮蔽，蒙蔽）、过（较轻的过失）、不善、恶**”。善人是行善之人，仁者是知道为何要行善的人。同样是善行，可以出自世俗趋名利的考虑，也可以处于真诚的心意，只有处于真诚的心意，才不会受外在环境的影响。

人生在世，很难判断到底**应该**怎么做才对，是否应该的判断，就来自**道义**。不问利害，而问该不该做，求心之所安。另外还要看礼的规范（包括**内心感受，对方期许，社会规范**）。一个人必须主动去[学习](#T12)，从学习中知道什么是真诚、善、仁，通过学习知道了自己为什么主动去做，教育就成功了，也代表这个人成熟了，可以独立了。

个人要自立，必须要有所专长，孔子认为行仁之人“君子不器”，也就是不能成为有特定用途的器具，如果拥有一技之长，不要只把自己当做这种专长的工具而已，还应该注意到君子对生命有全方位的要求；虽小道，必有可观者焉；致远恐泥，是以君子不为也（长期专注在一项技艺里，恐怕会陷入执着的困境）。

为人处世，要讲忠信。但是忠信之前，必须要做到君子贞【坚持大原则】而不谅【拘泥于小信】，也就是基本原则决不放弃，应用时可以有各种权衡。如何能找到复杂环境下权变之道，就还是回到真诚为本，择善固执的道路上去。

但是，人这一辈子肯定会犯错误，孔子对于改**过**自新的态度是“成事不说，遂事不谏，既往不咎”，能以未来的善行补救过往的错误，才是人生正途。论语中举《世说新语·栖逸》中戴氏兄弟的故事，“钟鼎山林，各有天性”，"人之过也，各与其党"来说明人犯错是由其性格决定的，只要将缺点改正，将优点发扬，就很容易走上正途。另外评价一个人不能单单从犯过的错误以及道德层面来评判（参见[管仲](#T2)一节）。

现代社会的人都希望能成名，成为闻人，但是孔子认为成名没有问题，但是还有更高一层的追求，就是成为达人。成名后能吃得开“在邦必闻，在家必闻”只是做到了“闻人”；如果要做到“达人”，就要“质直而好义，察言而观色，虑以下人”，做到这三点，人自然能通达。儒家讲究入世，因为善是我与别人之间适当关系的实现。尽管做官是具体的社会成就，可以作为志向，但是只达到孔子教育的一个层面。曾皙说的“暮春者，春服既成，冠者五六人，童子六七人，浴乎沂，风乎舞雩，咏而归”，就是即使没有达到自己做官的志向，照样可以配合天时地利人和，随时随地都可以快乐。曾皙这种志向显示出自由的生命情调，不受任何限制。如果志向被别人限制，就是身不由己。

儒家认为任何价值都是为了完成人性，对文学艺术作品来说本身有调和人性的作用就足够了，不需要强求文艺体现道德内涵，人们欣赏完文艺作品，心情受到调整，能够激发觉悟向善之心，然后再去行善即可。礼义信都是行善的范围。傅佩荣还对[儒家的羞耻文化和西方的罪恶文化](#T3)进行了分析对比。

人的生命由内在决定，“朝闻道，夕可死矣”，“寻孔颜乐处”，一定是从学习、理解、掌握人性的真相而来，知道人生的重点在内部在外，这时才是有源之水。人的尊严就是在这种“乐”中得到充分的肯定。论语中提到人走向正路的方法走向：“兴于诗，立于礼，成于乐”，所以多读《诗经》对人成长有好处，其一可以调节个人情绪，引发内心真诚，观察个人志节，沟通群中情感，纾解委屈怨恨；其二懂得如何侍奉父母与君主；其三可以广泛认识草木鸟兽名称，增长知识。“诵诗三百，授之以政，不达；使于四方，不能专对，虽多，亦奚以为？”

尧曰篇第二十，是古之帝王统治心法汇总，有志于此的管理者不妨仔细阅读。

《论语》以：“不知命，无以为君子也；不知礼，无以立也；不知言，无以知人也。”作为结束语，从最难的知命，到知礼，到最后的知言，正是人们择善固执，走向人生正途的一道道关口，钱穆先生每年都重读一遍论语，并重读重新每次都有新体会，最后的[论语摘录](#T19)，应当时时咏读，并用于日常生活中，这也是我辈重读经典并从中汲取营养的原因吧。

题外话，论语并非全都是人生体验的金句。其中有一句“唐棣之华，偏其反而。岂不尔思？室是远而”。子曰“未之思也，夫何远之有”，很有意思，如果真的很想思，又怎么会以路途遥远为理由不去相见呢？

### <a id="T2">管仲</a>

整部《论语》有六个人合乎行仁的资格，管仲排名第六。

孔子认为管仲不懂礼“管氏而知礼，孰不知礼？”。

- 管仲仕小白，孔子纵观全局评价，以一己之力造福天下百姓，与众人之间的适当关系得以实现，无异于行了大善【善是人与人之间适当关系的实现】。
- 管仲的过失，<3.22>，“管仲之器小哉！”，“管氏有三归，官事不摄，焉得俭？”，“管氏而知礼，孰不知礼？”
- 参见“一朝之忿”，“岂若匹夫匹妇之为谅也，自经于沟渎而莫之知也”
- 论语中六个人被称为仁者，“微子，比干，箕子，伯夷，叔齐，管仲”，谁都做过坏事，也都做过好事，重要的是怎样去“过而改之”，“成事不说，遂事不谏，既往不咎”，让自己的生命日日新，自强不息。
- 子夏曰“大德不逾闲，小德出入可也”
- 君子贞【坚持大原则】而不谅【拘泥于小信】：基本原则决不放弃，应用时可以有各种权衡

### <a id="T3">罪恶文化与羞耻文化</a>

文化的三个层次

- 器物层次：为人类生活方便，发明各种与衣食住行、经济发展、科技水平相关的东西。自然科学属于器物层次
- 制度层次：风俗习惯转变而成的法律规范，也成为制度。社会科学属于制度层次
- 理念层次：理想与观念。人为什么要活着？活着要完成的使命是什么。人文科学属于理念层次。人类如果没有理念，文化就没有任何实质性力量。

中国文化和西方文化不同，中国人谈耻感，西方人谈罪恶感，都是人性的黑暗面。西方罪恶感起源于人违反了和神的契约【旧约和新约】，佛洛依德说“很多人因为罪恶感而犯罪”，证明心理作用很强大。儒家所说的耻，界定在人与人之间，一个人的行为达不到社会所定的行为规则，就会觉得羞愧。别人的责问或者质疑，就会让人觉得不好意思。一个人从小到大，总会做出令自己不太满意、不好意思让别人知道的事情，这种状态就叫做羞耻感。

宇宙万物中，本质包含存在的，在西方人眼里就是上帝。上帝即西方哲学所谓“自因”。我们所见的一切事物都是“他因【由别的原因造成的】”，我们本身也是他因，因为是父母所生。因为人类属于他因，就有开始和结束，因而人的生命就有选择的可能。一旦选择就有可能选错，这就叫人性的弱点。因为人有自由，选错的可能性和实际性使人感到缺陷（不完美），这种缺陷，西方称为罪恶感，中国称为羞耻感。

儒家不忽略这种选择错误，如果对此避而不谈，只从正面看人，却又无法忽略负面阴暗部分，就会造成内外脱节，容易产生精神方面问题，人格分裂。儒家尝尝提到的“耻”，就是要主动要求自己，而不是等别人要求，这就是“耻辱”在自我要求中的积极作用。

儒家追求“知其不可而为之”，是承担社会责任。庄子“知其不可奈何而安之若命”，与世无争，接受命运，等待那个“势”，气势形成之后，就容易顺水推舟。其消极是外在消极，内心审美却上升到另一种境界，一点儿都不消极。谈儒家思想，积极的说法是“己欲立而立人，己欲达而达人”，也就是“行仁”。消极说法是“己所不欲，勿施于人”。一个进展通达，一个安稳立足。

“无为而治”，儒家强调“恭己正南面而已矣”【端坐在南方的王位上，有修德与尽职的责任，分层负责，而不是真正的无所事事。是以德为最高原则，强调道德，因为人性向善，所以可以无为而治】。道家的“无为而治”是说不要刻意为之，而是先了解大势所趋，然后顺其自然，以求水到渠成。

### <a id="T4">仁是什么</a>

仁，既是行仁，也就是“人之道”，是一辈子的事情，是动态的过程，是需要“择善固执”和“盖棺定论”的，子曰：若圣与仁，则吾岂敢，人在世的时候岂能说达到行仁的标准了呢？

人有专长，但是行仁是最难的事情，但是人要“君子不器”。

**仁**，有**人之性、人之道和人之成**三个层面。仁者无私心，所以很难。如果没有真诚的心意，不走在人生正途上，又没有完美人格，即使有礼和乐又有什么用呢。仁是三位一体的（即人之性，人之道，人之成）。求仁的过程，真诚是第一位的（思无邪），孝悌是真诚的出发点。

- 人之性：指**真诚**的心意以及向善的自觉力量
- 人之道：指人生正途或者**择善固执**
- 人之成：指**完美的人格**

孝心是求仁最易的方面，尽孝最难的是发自真诚的保持和颜悦色，所谓久病床前无孝子都是求仁之路上的考验。孝悌是与最亲近的人之间适当关系的实现，就是在行善，在修身，人人都能做到修身、齐家，社会风气良好，政治自然清明。

对父母尽孝：子曰：“事父母几谏，见志不从，又敬不违，劳而不怨”，“父母在，不远游，游必有方”，“三年无改于父之道，可谓孝矣”，“父母之年，不可不知也。一则以喜，一则以惧”。

- 樊迟问仁三次，“仁者先难而后获，可谓仁矣”；
- “爱人”；
- “居处恭，执事敬，与人忠。虽之夷狄，不可弃也”
- 刚、毅、木、讷【巧言令色，但是要真诚】，近仁
- 克【好胜】、伐【自夸】、怨、欲不行焉，可以为仁矣？子曰“可以为难矣，仁则吾不知也”
- 仁者必有勇，勇者不必有仁【行仁的人其勇气不在于对付别人，而在于克制自己，做自己的主人。“自胜者强”；勇敢的人很多，却不一定能行仁，因为他不一定可以走上人生正途】
- 能行五者于天下为仁矣：恭、宽、信、敏、惠。恭则不侮，宽则得众，信则人任焉，敏则有功，惠则足以使人
- 博学而笃志，切问而近思，仁在其中矣
- 如何行仁，“出门如见大宾，使民如承大祭【存敬，守礼，个人要遵守的外在礼节】。己所不欲，勿施于人【人与人之间的温情】。在邦无怨，在家无怨【群体和谐】”。“仁者，其言也仞【说话谨慎】”

### <a id="T6">友是什么</a>

观察人的方法要看过去现在和未来的表现（视其所以，观其所由，察其所安），重点在做在求的过程中。关于想和做的问题，子曰“先行其言，而后从之”。

对不同意见和观点要包容，子曰“道不同，不相为谋”，“攻乎异端，斯害也已”。

- 和新朋友交往时要做到“敬而无失，与人恭而有礼，四海之内皆兄弟也”。
- 子贡问友，子曰“忠告而善道之，不可则止，勿自辱焉”
- 曾子曰“君子以文会友，以友辅仁”
- 狷者【有所不为之人，易找】->中行【当狂则狂，当狷则狷，最难】->狂者【进取之人，比较难】
- 切切偲偲，怡怡如也，可谓士矣；朋友切切偲偲，兄弟怡怡【朋友是自己选择的，如果不能经常切磋，互相期许，岂不是浪费时间；兄弟姐妹是血缘关系，吵不散，分不开，和睦第一】
- 益者三友，友直，友谅，友多闻
- 损者三友，友便辟，友善柔，友便佞

### <a id="T8">政是什么</a>

- 问政【政治】，民信>足食>足兵；

- “居之无倦，行之以忠【坚持原则，从不倦怠，为官的实质】”

- 子路问政，子曰“先之劳之”“无倦”。一个人的能力不在于能得到什么，在于能承受失去什么，是一种耐力。凡事能稳住阵脚，面对变化，内心有一股力量支撑自己。外求一个目标相对容易，难的是内守一种稳定的力量。

- “先有司，赦小过，举贤才”，仲弓曰“焉知贤才而举之”，子曰“举尔所知，尔所不知，人其舍诸？”【只要举尔所知，至于不认识的人才，只要够优秀，别人也不会错过他们】

- 其身正，不令而行；其身不正，虽令不从

- 施政顺序，庶矣->富之->教之【受了教育之后，人必须自行努力】

- 叶公问政，子曰“近者说，远者来”

- 子夏问政，子曰“无欲速，无见小利。欲速则不达，见小利则大事不成”

- 治国良策：知及之，仁能守之，庄以莅之，动之以礼

### <a id="T9">善是什么</a>

- 善人行善，但不知“人性向善”的道理，所以可能不明白“为何”非行善不可；善人未必为善牺牲生命；仁者知道人性向善，真诚而发择善固执，必要时可以舍身取义，杀身成仁。
- 行善有成的人，有恒【相对于择善的“固执”而言】到一定程度，即可成为善人。但是如果注意力外求得失，就很难做到有恒了。“强行者有志”。“深则厉，浅则揭【出自《诗经·邶风·匏有苦叶》，要想过河就可能弄湿衣服，如果犹豫不决，将一事无成。】”
- 子张问善人之道，子曰“不践迹，亦不入于室【不随俗从众，但是修养也没有达到最高境界】”
- 善人为邦百年，亦可以胜残去杀矣
- 善人教民七年，亦刻意即戎矣
- 仁和义都是善的具体表现

### <a id="T10">士是什么</a>

- 行已有耻，使于四方，不辱使命，可谓士矣
- 宗族称孝焉，乡党称悌焉
- 言必行，行必果，硁硁然小人哉
- 切切偲偲，怡怡如也，可谓士矣；朋友切切偲偲，兄弟怡怡
- 士而怀居，不足以为士矣【读书人不能死读书，留恋安逸的生活；应该以天下为己任，如果有机会，就要实现自己的理想；怀居则愧为读书人；人总是有想做而没机会做的事情，到了一定年龄，要考虑调整自己的方向，而不是一味追求安逸和享乐】
- 知【臧武仲之知】、不欲【公绰之不欲】、勇【卞庄子之勇】、艺【冉求之艺】，文之以礼乐【外加礼乐教化】，亦可以为成人矣【才可称为成人【理想的人】】。

- 为政“用之则行，舍之则藏”

### <a id="T12">学习</a>
孔子的学有三个特点：一、老老实实学习知识与技术；二、基于学得，有个人体会；三、将体会应用到个人行为并作出改变。可见其学是全方位的、主动的学习与成长过程，同时“学则不固（博学多闻就不会胶柱鼓瑟）”也是做人处事的道理。

自己努力学习，并于朋友相互呼应探讨，在学习过程或者低谷中，可以坦然自处。人生在世，学以致用才是关键。化天命为自己能掌控的命运，通过求仁的过程，实现自己的使命。

好学有三个步骤，一、降低物质享受欲望，在言行上；二、在言行上磨练及改善自己；三、虚心向良师请教，走上仁的道路。

学习过程有三种：蚂蚁搬食，蜜蜂酿蜜，老鹫抟云；类似于孔子说的知之、好之、乐之；耶鲁大学校长也类比与：来学习、来理解、来品味。

学习的四个阶段：共学，适道，立，权。

《论语》中提到四次“好学”，一次说君子，一次说孔子自己，一次说颜渊，还有一次说孔文子。

君子“驷不及舌。文犹质也，质犹文也。虎豹之鞟犹犬羊之鞟”，君子除了淳朴之外，还需要文饰，需要通过学习与理解做到。言之无文，行之不远。“博学于文，约之以礼”

如何增进德行【崇德】“主忠信，徙义，崇德也”。“先事后得，非崇德与？”

如何消除积怨【修慝】“攻其恶，无攻人之恶，非修慝与？”

如何辨别迷惑【辩惑】“爱之欲其生，恶之欲其死，既欲其生又欲其死，是惑也”。“一朝之忿，忘其身以及其亲，非惑与？”【韩信胯下之辱是一例；管仲仕小白】

“君君臣臣父父子子”，讨论了“名和实”的关系。除了有君之名还要有君之实【也就是做到君臣父子应该做的，而不是一味要求别人对自己】；事君“勿欺也，而犯【进谏】之”；事君，敬其事而后其食；陈力就列，不能者止；

待人上做到不迁怒，“已所不欲，勿施于人”，克己做到不贰过，“日新又新”。好学，以德行为首要目标，而不只是学习各种知识，学习之后要不断实践，做中练。

学习的重要性，六言六弊：

- 好仁不好学，其蔽也愚；好知不好学，其蔽也荡；好信不好学，其蔽也贼；好直不好学，其蔽也绞；好勇不好学，其蔽也乱；好刚不好学，其蔽也狂。

儒家重视理性，强调学习，也肯定良知。但是良知很圆满就有点过了。心与仁不同，仁是人之道，是人生正途。心做自觉的选择，可以选择行仁，也可以选择不行仁，心选择行仁如此困难，但是还是必须行仁，是因为“人性向善”，除了走向人生正途，别无出路。因此，心与仁结合，需要长期修炼磨合。

### <a id="T16">什么是礼</a>

人我互动层面第一个价值是礼法，法律偏消极，礼仪则比较内化主动；第二个价值是情义，其对象一定是认识的人，而且必定要有所牺牲，花时间、精力、金钱。子路说“愿车马衣裘，与朋友共，敝之而无憾”；颜渊曰“愿无伐劳，无施劳”；子曰“老者安之，朋友信之，少者怀之”。

### <a id="T17">什么是真诚</a>

直，指真诚而正直，并不等于客观的法律规定，真诚是正直的前提。“大义灭亲”违背常情，是当权政治领袖用来欺骗百姓的说法。儒家思想提倡“父为子隐，子为父隐，直在其中矣”，儒家思想可贵之处在于它是以人的正常情绪，情感为出发点。如果与此相违背，就不和情理【如何判断是正常情绪呢，靠真诚，择善固执，修身】

人生在世，想要简简单单快快乐乐，必须越单纯越好，人际关系要都清清楚楚，不可沽名钓誉，比如“或乞醯焉，乞诸其邻而与之”这种行为。生活上，与人有嫌隙时，就要把话说清楚，如果表面装作不在乎，实则心里很在意，“匿怨而友其人”，对解决问题不利。

“质胜文则野，文胜质则史。文质彬彬，然后君子”；具体操作上有两种表现，其一是表现纹饰时保持质朴之心；一是该文雅时文雅，该朴实时朴实。二者依照环境及对象而定。

孔子自我期许有三件事：“默而识之，学而不厌，诲人不倦”。

孔子从不懈怠的四件事：“德之不修，学之不讲，闻义不能徙，不善不能改”

孔子人生内求追四件事：“至于道，据于德，依于仁，游于艺”。但是子夏曰“虽小道，必有可观者焉；致远恐泥，是以君子不为也”

孔子以慎重态度对待的三件事：“齐、战、疾”

孔子对自己的评价“三忘：忘食，忘忧，忘老”：“其为人也，发愤忘食，乐以忘忧，不知老之将至云尔”。

孔子教学四项重点：“文、行、忠、信”；两大原则：“博我以文，约我以礼”。

教育启发学生，必做到：“不愤不启，不悱不发。举一隅不以三隅反，则不复也”

阻碍我们进步，有恒的三种毛病：“亡而为有，虚而为盈，约而为泰”。

孔子可以作为情商培养的榜样：“子温而厉，威而不猛，恭而安”

孔子为人几个方面，都表现为“约”：

- 谨慎：“齐、战、疾”
- 罕言：“子罕言利与命与仁”
- 不语：“子不语怪力乱神”
- 子绝四：毋意，毋必，毋固，毋我

如何实现克己复礼，“非礼勿视，非礼勿听，非礼勿言，非礼勿动”。
